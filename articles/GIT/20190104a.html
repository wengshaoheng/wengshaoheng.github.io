Git的标签使用方法

1. Create Tag:

1) git tag tag-id e.g. git tag v1.0 2) git tag -a tag-id -m "comment" e.g. git tag -a v1.0 -m "release version 1.0" 3) git push <origin> --tags e.g. git push --tags

2. Delete Tag:

1) git tag -d tag-id e.g. git tag -d v1.0 2) git push <origin> :refs/tags/tag-id e.g. git push :refs/tags/v1.0
